SI AÚN NO HA INSTALADO SGHC
===========================

Para instalar SGHC, simplemente inserte el primer disco en su unidad de disco flexible y ejecute el programa INSTALAR.EXE, a continuación siga las instrucciones que aparecen en pantalla e inserte el resto de discos cuando le sea solicitado.

INICIANDO SGHC
==============

Cuando inicie SGHC se le solicitará el ingreso de una clave, si usted no ha configurado clave alguna, cancele este diálogo y la ejecución del programa continuará normalmente.


VER LOS DATOS DE EJEMPLO
========================

Podrá ver datos de ejemplo abriendo el archivo DEMO\DEMO2.DBP desde el menú Archivo|Abrir.


RECOMPILAR SGHC
===============
Esta versión del SGHC no trae el código fuente, y por lo tanto no puede ser recompilada.

DESINSTALAR SGHC
================
Para desinstalar SGHC, haga clic en el menú Inicio, escoja Programas | Horarios para Colegio 1.2 | Desinstalación de Horarios para Colegio 1.2 y confirme la operación.

Por seguridad, la base de datos no se desinstala automáticamente, por lo que si desea eliminar completamente el SGHC, las carpetas SRC y DAT deberán borrarse manualmente.

PROBLEMAS CONOCIDOS
===================
Año 2000:  En principio el sistema no depende de las fechas, por lo que es casi imposible que surja un problema de "año 2000".  Si en las fechas cortas se muestra el año únicamente con dos dígitos, puede cambiar la configuración en Panel de control | Configuración regional | Fecha, para que se muestre el año con cuatro dígitos.
